The front controller design pattern is used to provide a centralized request handling mechanism so that all requests will be handled by a single handler. This handler can do the authentication/ authorization/ logging or tracking of request and then pass the requests to corresponding handlers. Following are the entities of this type of design pattern.

  • Front Controller - Single handler for all kinds of requests coming to the application (either web based/ desktop based).

  • Dispatcher - Front Controller may use a dispatcher object which can dispatch the request to corresponding specific handler.

  • View - Views are the object for which the requests are made.

Implementation

We are going to create a FrontController and Dispatcher to act as Front Controller and Dispatcher correspondingly. HomeView and StudentView represent various views for which requests can come to front controller.

FrontControllerPatternDemo, our demo class, will use FrontController to demonstrate Front Controller Design Pattern.

Step 1

Create Views.

HomeView.java

1public class HomeView {
2   public void show(){
3      System.out.println("Displaying Home Page");
4   }
5}

StudentView.java

1public class StudentView {
2   public void show(){
3      System.out.println("Displaying Student Page");
4   }
5}

Step 2

Create Dispatcher.

Dispatcher.java

 1public class Dispatcher {
 2   private StudentView studentView;
 3   private HomeView homeView;
 4   
 5   public Dispatcher(){
 6      studentView = new StudentView();
 7      homeView = new HomeView();
 8   }
 9
10   public void dispatch(String request){
11      if(request.equalsIgnoreCase("STUDENT")){
12         studentView.show();
13      }
14      else{
15         homeView.show();
16      }
17   }
18}

Step 3

Create FrontController

FrontController.java

 1public class FrontController {
 2
 3   private Dispatcher dispatcher;
 4
 5   public FrontController(){
 6      dispatcher = new Dispatcher();
 7   }
 8
 9   private boolean isAuthenticUser(){
10      System.out.println("User is authenticated successfully.");
11      return true;
12   }
13
14   private void trackRequest(String request){
15      System.out.println("Page requested: " + request);
16   }
17
18   public void dispatchRequest(String request){
19      //log each request
20      trackRequest(request);
21      
22      //authenticate the user
23      if(isAuthenticUser()){
24         dispatcher.dispatch(request);
25      }
26   }
27}

Step 4

Use the FrontController to demonstrate Front Controller Design Pattern.

FrontControllerPatternDemo.java

1public class FrontControllerPatternDemo {
2   public static void main(String[] args) {
3   
4      FrontController frontController = new FrontController();
5      frontController.dispatchRequest("HOME");
6      frontController.dispatchRequest("STUDENT");
7   }
8}

Verify the output.
